При создании смет программа работает с набором баз всех официальных нормативов, включенных в федеральный реестр сметных нормативов (ФЕРы, ТЕРы). Это является гарантией успешного обоснования стоимости сметы.

Есть возможность подгрузить ведомственные сборники (Минэнерго, РЖД, реставрационные и прочие), базы НЦС/НЦКР.

Программа содержит полную базу индексов Минстроя, Минэкономразвития, дополнительно подгружаются региональныеиндексы по статьям затрат, ведомственные индексы, сборники текущих цен.

Можно создавать фирменные сборники баз данных для собственных расчетов.В программу можно подгрузить все официальные нормативы, включенные в федеральный реестр сметных нормативов (ФЕРы, ТЕРы).

Есть возможность подгрузить ведомственные сборники (Минэнерго, РЖД и прочие).

Можно создавать фирменные сборники баз данных для собственных расчетов.

Подробнее

Программа позволяет осуществить любой метод расчета смет

- Базисно- индексный
- Ресурсный
- Смешанный и т.д.

При расчете сметы базисно-индексным способом индексы Минстроя в программе обновляются по мере их выхода.Индекс можно начислить на расценку, на раздел или на итог по смете. В программе есть возможность начислитькоэффициент на индекс, усложняющие коэффициенты. Накладные расходы и сметная прибыль в программе начисляются автоматически.

При расчете сметы ресурсным способом существует удобный механизм расценивания. При использовании этой возможностипрограмма автоматически расценит материалы, механизмы, трудозатраты рабочих и механизаторов.

Подробнее

Программа содержит функции, которые позволяют сделать работу с ресурсами удобнее и быстрее. Все изменения внутри расценки наглядно отображаются в виде цветовых и буквенных меток. Ресурс можно пометить как ресурс заказчика, возвратный материал, металлолом, мусор, после чего получить обобщенные данные о массе мусора и металлолома по смете, стоимости возвратных материалов и ресурсах заказчика.

Существует быстрый автоматический подбор материалов из нормативной базы для замены одного материалов на другой.

Очень удобной функцией программы является редактор ресурсов. позволяет осуществлять групповое редактирование цен на ресурсы, групповые замены. Из него также можно распечатать ведомости используемых ресурсов. В редакторе существует система фильтров, позволяющая выбрать только те ресурсы, с которыми работаем в настоящий момент (материалы, механизмы, рабочие, механизаторы, оборудование и т.д. ) . Группировка ресурсов происходит по различным заданным критериям.

Подробнее

Расчет объемов выполняемых работ в программе «Адепт: Смета».

В программе есть отдельный модуль, который позволяет просчитывать объемы разных работ, составлять ведомость объемов работ, привязывать её к расценкам для загрузки в сметные программы.

В данном модуле более 150 формул, по которым можно составить ведомость, также есть возможность заносить в базу свои формулы. Формулы наглядно визуализируются в чертежах. Это дает возможность учитывать переменные в формуле грамотно.

Подробнее

В программе автоматизирована работа с ФСНБ 2022

- Загрузка данных из баз ФГИС ЦС (загрузка прейскурантов для ресурсов, отсутствующих в ФСБЦ и ФСЭМ)
- Единые базы, совмещающие цены на ресурсы из ФГИС ЦС и цены из ФСБЦ/ФСЭМ с учетом индексов ФГИС ЦС
- Удобный механизм расценивания
- Удобный механизм замены материала
- Автоматизация надбавок
- Автоматизированный пересчет смет из ФСНБ-2020 в ФСНБ-2022

Подробнее

Программа содержит упорядоченную систему папок и подпапок для хранения необходимых документов по стройке и договоров прямо внутри программы.

Все документы, приложенные к стройкам или договорам легко переносятся в другие программы серии Адепт.

Договора можно формировать по шаблону или создавать свои шаблоны договоров с автоматическим проставлением данных по партнерам.

Подробнее

В программе есть возможность создавать калькуляции на основе собственных расценок и нормативов, а также с применением собственных материалов и механизмов.

Можно подгрузить калькуляции из Excel.

Готовую калькуляцию можно вывести на печать.

Подробнее

Дополнительный модуль «Адепт: Перебазировка» позволяет создавать в расчеты на перебазировку строительной техники.

После выбора схемы перебазирования открывается окно расчетов, позволяющее полностью автоматизировать процесс определения стоимости перебазировки. Необходимо лишь ввести всю требующуюся информацию в соответствующие поля.

При выборе машины с помощью поиска по базе ЭМ, программой автоматически заполняются большинство полей. После ввода всех данных и сохранения результат калькуляции отобразится в расчете на экране.

Подробнее

В программе «Адепт: Смета» есть возможность создавать акты выполненных работ различными способами. Форма вывода акта на печать может быть разной, программа предлагает варианты.

В зависимости от типа закрытия акта, объемы выполненных работ проставляются автоматически, или пользователь их задает вручную, ориентируясь на доступный остаток. Для печати доступно множество форм. В программе легко и быстро, нажав на кнопку, можно создать акт по форме КС-2. Можно выбрать любой тип закрытия акта (процент от сметы, раздела или остатка , по объемам, полное закрытие и т.д.) Отображения данных по материалу для списания могут быть учтены в акте и автоматически будут отправлены в М-29.

Если есть необходимость внести коррективы в акте КС-2, программа позволяет создать независимый акт.

Подробнее

Несовпадения фактически закупленных материалов с данными по смете-акту отображаются в форме М-29.

Форму М-29 можно создать в программе, нажав на одну кнопку и выбрав галочками необходимые сметы. Форма состоит из двух частей.

В 1 части формы отчета по М-29 можно увидеть нормативную потребность в материалах по каждой расценке по выбранным сметам.

Во 2 части для каждого месяца кроме расхода по производственным нормам показывается фактический расход. При этом отображается разница (экономия и перерасход) по каждому материалу.

Подробнее

Часто возникает необходимость сравнения двух смет между собой (в процессе проверки сметы при сравнении разных версий сметы, сравнении с эталонной сметой, смет от двух разных сметчиков, смет, полученных с использованием разных сметных программ).

Для реализации функции сравнения двух смет между собой в программе разработан Контроллер смет, калькуляций и актов, который позволяет произвести построчное сравнение любых смет, калькуляций и актов друг с другом.

В контроллере есть возможность экспертизы индексов, накладных расходов и сметной прибыли, а также проверка на соответствие баз данных

Подробнее

В программе «Адепт: Смета» есть возможность создавать пояснительную записку.

Пояснительную записку можно создать по локальной смете или по сводному сметному расчету.

Подробнее

Программа «Адепт: Смета» позволяет создавать расчет по сборникам НЦС (Укрупненные нормативы цены строительства) для планирования инвестиций (капитальных вложений), оценки эффективности использования средств направляемых на капитальные вложения и подготовки технико-экономических показателей в задании на проектирование.

Реализованы различные механизмы расчета в зависимости от условий показателя объекта, в том числе несколько вариантов интерполяции.

Формула расчета интерполяции автоматически выводится в окне расценки при вводе необходимой мощности.

Подробнее

Дополнительный модуль «Адепт: СЭД» позволяет организовать любую структуру хранения смет и организовать работу в едином информационном пространстве любого количества сметчиков, работающих в разных филиалах.

У каждого документа могут быть свои атрибуты, автор смет, проверяющий этой сметы и ее статус проверки.

Информация хранится в едином пространстве, синхронизируя между собой географически распределенные подразделения компании, обеспечивая надежную систему хранения, обработки и последующей передачи смет и иных документов с возможностью их проверки.

    Можно подгрузить калькуляции из Excel.

    Готовую калькуляцию можно вывести на печать.

  • Расчет стоимости перебазировки

    • Дополнительный модуль «Адепт: Перебазировка» позволяет создавать в расчеты на перебазировку строительной техники.

    После выбора схемы перебазирования открывается окно расчетов, позволяющее полностью автоматизировать процесс определения стоимости перебазировки. Необходимо лишь ввести всю требующуюся информацию в соответствующие поля.

    При выборе машины с помощью поиска по базе ЭМ, программой автоматически заполняются большинство полей. После ввода всех данных и сохранения результат калькуляции отобразится в расчете на экране.

  • Формы КС-2, КС-3. Акты выполненых
    работ

    • В программе «Адепт: Смета» есть возможность создавать акты выполненных работ различными способами. Форма вывода акта на печать может быть разной, программа предлагает варианты.

    В зависимости от типа закрытия акта, объемы выполненных работ проставляются автоматически, или пользователь их задает вручную, ориентируясь на доступный остаток. Для печати доступно множество форм. В программе легко и быстро, нажав на кнопку, можно создать акт по форме КС-2. Можно выбрать любой тип закрытия акта (процент от сметы, раздела или остатка , по объемам, полное закрытие и т.д.) Отображения данных по материалу для списания могут быть учтены в акте и автоматически будут отправлены в М-29.

    Если есть необходимость внести коррективы в акте КС-2, программа позволяет создать независимый акт.

    Подробнее
  • Форма М-29

    • Несовпадения фактически закупленных материалов с данными по смете-акту отображаются в форме М-29.

    Форму М-29 можно создать в программе, нажав на одну кнопку и выбрав галочками необходимые сметы. Форма состоит из двух частей.

    В 1 части формы отчета по М-29 можно увидеть нормативную потребность в материалах по каждой расценке по выбранным сметам.

    Во 2 части для каждого месяца кроме расхода по производственным нормам показывается фактический расход. При этом отображается разница (экономия и перерасход) по каждому материалу.

    Подробнее
  • Экспертиза смет. сравнение версий
    смет и актов

    • Часто возникает необходимость сравнения двух смет между собой (в процессе проверки сметы при сравнении разных версий сметы, сравнении с эталонной сметой, смет от двух разных сметчиков, смет, полученных с использованием разных сметных программ).

    Для реализации функции сравнения двух смет между собой в программе разработан Контроллер смет, калькуляций и актов, который позволяет произвести построчное сравнение любых смет, калькуляций и актов друг с другом.

    В контроллере есть возможность экспертизы индексов, накладных расходов и сметной прибыли, а также проверка на соответствие баз данных

    • Программа «Адепт: Смета» позволяет создавать расчет по сборникам НЦС (Укрупненные нормативы цены строительства) для планирования инвестиций (капитальных вложений), оценки эффективности использования средств направляемых на капитальные вложения и подготовки технико-экономических показателей в задании на проектирование.

    Реализованы различные механизмы расчета в зависимости от условий показателя объекта, в том числе несколько вариантов интерполяции.

    Формула расчета интерполяции автоматически выводится в окне расценки при вводе необходимой мощности.

Серверная ЭВМ (Аппаратная часть)

Системные требования для 32-битной версии
Для инсталляции Программы Лицензиат должен иметь исправную ЭВМ, отвечающую следующим требованиям:
• IBM-совместимый компьютер с процессором не ниже Pentium, частота не ниже 1 ГГц
• ОЗУ – не менее 512 Мб, если операционная система Windows NT, 2000, ХР, 2003
• ОЗУ – не менее 1 Гб, если операционная система Windows 7, Windows Vista
• если есть необходимость работы с MS Office, то необходим MS Office версии не ранее 2007 г
• Свободное место на жестком диске – не менее 3 Гб
• Минимальное разрешение монитора 1024х768.
